# 개인 맞춤형 스마트 학습 플랫폼

**AI 기반 개인화 학습 경험**을 제공하는 혁신적인 에듀테크 플랫폼으로, 학습자의 수준과 선호도를 분석하여 최적의 학습 경로를 제안하고 실시간 피드백을 제공합니다.

## 🎯 프로젝트 배경

### 교육 현장의 문제점
- **획일화된 교육**: 개인별 학습 속도와 스타일의 차이 무시
- **진도 관리의 어려움**: 학습자의 실제 이해도 파악 한계
- **동기 부여 부족**: 지속적인 학습 동기 유지의 어려움
- **실시간 피드백 부재**: 즉각적인 학습 성과 확인 불가

### 솔루션 접근법
- **AI 학습 분석**: 머신러닝 기반 개인별 학습 패턴 분석
- **적응형 컨텐츠**: 실시간 난이도 조절 및 맞춤 학습 자료 제공
- **게이미피케이션**: 학습 동기를 높이는 포인트/배지 시스템
- **소셜 러닝**: 학습자 간 상호작용 및 협업 학습 지원

## 🛠️ 기술 스택

### Frontend (Vue.js 3 + TypeScript)
```typescript
// 학습 진도 추적 인터페이스
interface LearningProgress {
  userId: string;
  courseId: string;
  completedLessons: string[];
  currentLesson: string;
  difficultyLevel: 1 | 2 | 3 | 4 | 5;
  timeSpent: number; // 분 단위
  comprehensionScore: number; // 0-100
  lastAccessed: Date;
}

// AI 추천 학습 경로
interface RecommendedPath {
  pathId: string;
  estimatedDuration: number;
  difficulty: number;
  prerequisites: string[];
  learningObjectives: string[];
  adaptiveContent: boolean;
}
```

### Backend (Node.js + Express)
- **학습 분석 엔진**: TensorFlow.js 기반 학습 패턴 분석
- **실시간 통신**: Socket.io를 활용한 라이브 세션
- **인증 시스템**: JWT + OAuth 2.0 (Google, Kakao)
- **데이터베이스**: PostgreSQL + Redis (캐싱)

### AI/ML 구성 요소
- **학습 스타일 분석**: K-means 클러스터링으로 학습자 그룹화
- **추천 시스템**: Collaborative Filtering + Content-based 하이브리드
- **난이도 예측**: Gradient Boosting으로 최적 난이도 산정
- **성취도 예측**: LSTM으로 학습 성과 예측

## 📋 핵심 기능

### 1. AI 기반 개인화 학습 경로
```python
# 학습 경로 추천 알고리즘
class LearningPathRecommender:
    def __init__(self):
        self.model = self.load_trained_model()
    
    def recommend_path(self, user_profile, learning_goal):
        # 사용자 학습 스타일 분석
        learning_style = self.analyze_learning_style(user_profile)
        
        # 현재 지식 수준 평가
        knowledge_level = self.assess_knowledge_level(user_profile)
        
        # 최적 학습 경로 생성
        optimal_path = self.generate_path(
            learning_style, 
            knowledge_level, 
            learning_goal
        )
        
        return optimal_path
```

### 2. 실시간 학습 분석 대시보드
- **학습 진도 시각화**: Interactive 차트로 진도율 및 성취도 표시
- **약점 분석**: 틀린 문제 패턴 분석 및 보완 학습 제안
- **시간 관리**: 학습 시간 추적 및 최적 학습 스케줄 제안
- **성과 예측**: AI 기반 목표 달성 가능성 예측

### 3. 적응형 학습 컨텐츠
- **동적 난이도 조절**: 실시간 이해도에 따른 문제 난이도 자동 조정
- **멀티미디어 학습**: 텍스트, 동영상, 인터랙티브 시뮬레이션
- **마이크로러닝**: 5-10분 단위의 집중력 최적화 학습 모듈
- **반복 학습**: 망각 곡선을 고려한 복습 스케줄링

### 4. 소셜 러닝 & 협업
- **스터디 그룹**: 유사 수준 학습자 자동 매칭
- **실시간 Q&A**: 동료 학습자 및 멘토와 즉시 소통
- **학습 챌린지**: 그룹 대항 학습 경쟁 및 협업 과제
- **피어 리뷰**: 학습자 간 상호 평가 및 피드백
